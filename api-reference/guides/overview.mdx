## Particle's Onboarding SDKs

Particle offers a suite of SDKs to simplify onboarding through **Wallet Abstraction** and **Account Abstraction**. These tools help developers embed secure smart wallets, enable social logins, and manage smart accountsâ€”all from a unified, customizable interface.

## Quickstarts

Particle Connect allows you to implement social logins and account abstraction into your dApp with a single SDK, while Particle Auth provides a flexible package for integrating social logins and account abstraction into dApps.
