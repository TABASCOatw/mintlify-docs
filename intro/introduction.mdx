# Particle Network Documentation

Particle Network enables **Chain Abstraction** through **Universal Accounts**, allowing users to interact with multiple blockchains via a single account and unified experience.

Whether you're getting started or building advanced multi-chain apps, this documentation helps you:

- Integrate Universal Accounts and Social Logins
- Understand how Chain Abstraction works
- Use our SDKs and APIs to power your dApp
